# Useful Emacs Tidbits

## General

- Follow any keycombo up with *C-h* and you will git a help window
  describing all of the things you can subsequently do. For example,
  *C-x* followed by *C-h* will result in a help window describing
  every follow-up key combo that can be used. This helps a lot with
  discoverability and remembering key combos that are not used often.

## Bookmarks

Bookmarks allow for easy navigation between different files. It lets
you save your spot in a particular file and navigate to it later in
the editing session or even on a cold start of a new emacs session.

- Root keybinding: *C-x r*
  - Bookmark Set: *C-x r m*
  - Bookmark Jump: *C-x r b*
